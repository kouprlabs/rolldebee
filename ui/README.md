# Rolldebee UI

## Getting Started

Install [Node.js](https://nodejs.org/en/download), preferrably with [NVM](https://github.com/nvm-sh/nvm).

### Build and Run

Install dependencies:

```shell script
npm install
```

Run for development:

```shell script
npm run dev
```

Build Docker image:

```shell
docker build -t rolldebee/ui .
```
